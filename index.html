1. Pendahuluan HTML
Apa itu HTML?

HTML (HyperText Markup Language) adalah bahasa markup standar untuk dokumen yang dirancang untuk ditampilkan di browser web. HTML dapat dibantu oleh teknologi seperti Cascading Style Sheets (CSS) dan bahasa scripting seperti JavaScript.

Struktur Dasar HTML

Setiap dokumen HTML dimulai dengan <!DOCTYPE html> untuk mendeklarasikan tipe dokumen dan diapit dalam tag <html>.

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <title>Halaman HTML Pertamaku</title>
</head>
<body>
    <h1>Halo, Dunia!</h1>
    <p>Ini adalah struktur dasar HTML.</p>
</body>
</html>

Bagian <head> berisi meta-informasi tentang dokumen HTML, sedangkan bagian <body> berisi semua konten yang terlihat oleh pengguna.

2. Format Teks dan Judul
Judul (h1 hingga h6)

Judul mendefinisikan struktur konten Anda, dengan <h1> menjadi yang paling penting dan <h6> yang paling tidak penting.

<h1>Judul Utama</h1>
<h2>Judul Bagian</h2>
<h3>Judul Sub-bagian</h3>
<h4>Judul Lebih Kecil</h4>
<h5>Judul Bahkan Lebih Kecil</h5>
<h6>Judul Paling Tidak Penting</h6>
Paragraf dan Pemformatan Dasar

Tag <p> mendefinisikan sebuah paragraf. Anda dapat menerapkan berbagai gaya:

  • Teks tebal menggunakan <b> atau <strong> (untuk kepentingan semantik).
  • Teks miring menggunakan <i> atau <em> (untuk penekanan semantik).
  • Teks bergaris bawah menggunakan <u>.
  • Garis baru menggunakan <br>.
  • Garis horizontal menggunakan <hr>.
<p>Ini adalah paragraf normal.</p>
<p><b>Teks ini tebal.</b></p>
<p><i>Teks ini miring.</i></p>
<p><u>Teks ini bergaris bawah.</u></p>
<p>Ini adalah paragraf <br> dengan baris baru.</p>
<hr>
<p><strong>Perhatian:</strong> Teks ini penting.</p>
<p>Saya <em>sangat</em> lapar.</p>
Superskrip dan Subskrip

Untuk rumus matematika atau kimia:

  • Superskrip (misalnya, O2) menggunakan <sup>.
  • Subskrip (misalnya, CO2) menggunakan <sub>.
<p>Oksigen: O<sup>2</sup></p>
<p>Karbon Dioksida: CO<sub>2</sub></p>
Kutipan
  • Kutipan panjang menggunakan <blockquote>.
  • Kutipan singkat sebaris menggunakan <q>.
<blockquote>
    <i>Sekali teman tetap teman.</i>
</blockquote>
<p>Kutipan favorit saya adalah <q>man jadda wa jadda</q>.</p>
Singkatan dan Akronim
  • Singkatan menggunakan <abbr> (menampilkan tooltip saat kursor diarahkan).
  • Akronim menggunakan <acronym> (mirip dengan abbr, untuk akronim yang dibaca sebagai kata).
<p>Saya sekolah <abbr title="Sekolah Dasar">SD</abbr>.</p>
<p>Saya kuliah di <acronym title="Universitas Nurul Jadid">UNUJA</acronym>.</p>
Referensi dan Definisi
  • Kutipan untuk karya (buku, film) menggunakan <cite>.
  • Mendefinisikan istilah menggunakan <dfn>.
<p>Saya suka membaca buku <cite>harry potter</cite>.</p>
<p><dfn>SD</dfn> adalah singkatan dari sekolah dasar.</p>
Teks yang Dihapus dan Disisipkan
  • Teks yang dihapus menggunakan <del> (biasanya dicoret).
  • Teks yang disisipkan menggunakan <ins> (biasanya digarisbawahi).
<p>Saya suka <del>seblak</del> <ins>mie ayam</ins>.</p>

3. Daftar (List)
Daftar Berurutan (Ordered List - OL)

Item-item secara default diberi nomor.

<ol>
    <li>Denny Caknan</li>
    <li>NDX AKA</li>
</ol>
  1. Denny Caknan
  2. NDX AKA
Daftar Tidak Berurutan (Unordered List - UL)

Item-item secara default diberi tanda bullet. Dapat berisi daftar bersarang.

<ul>
    <li>Denny Caknan</li>
    <li>NDX AKA
        <!-- Daftar bersarang -->
        <ul>
            <li>Sinarengan</li>
            <li>Sanes</li>
        </ul>
        <ol>
            <li>Sinarengan</li>
            <li>Sanes</li>
        </ol>
    </li>
</ul>
  • Denny Caknan
  • NDX AKA
    • Sinarengan
    • Sanes
    1. Sinarengan
    2. Sanes
Daftar Definisi (Definition List - DL)

Digunakan untuk istilah dan definisinya.

<dl>
    <dt>UNUJA</dt>
    <dd>UNUJA adalah kampus yang ada di Pondok Pesantren Nurul Jadid.</dd>
</dl>
UNUJA
UNUJA adalah kampus yang ada di Pondok Pesantren Nurul Jadid.

<a href="list.html">Halaman List</a>

5. Atribut HTML
Atribut Umum

Atribut memberikan informasi tambahan tentang elemen HTML. Atribut ditentukan dalam tag pembuka dan biasanya datang dalam pasangan nama/nilai seperti nama="nilai".

  • href: Menentukan URL untuk tautan.
  • align: (Usang, gunakan CSS) Menentukan perataan konten.
  • title: Memberikan informasi tambahan tentang elemen (tooltip saat kursor diarahkan).
  • src: Menentukan jalur ke file gambar, video, atau audio.
  • width, height: Menentukan dimensi media.
  • alt: Memberikan teks alternatif untuk gambar (untuk aksesibilitas).
  • controls: Menambahkan kontrol pemutaran untuk media.
  • autoplay: Memulai pemutaran media secara otomatis.
  • loop: Mengulang pemutaran media.
  • poster: Menentukan gambar yang muncul sebelum video diputar.
  • preload: Menentukan apakah dan bagaimana browser harus memuat media saat halaman dimuat.
Contoh Atribut

Contoh: href dan target

<a href="https://www.google.com" target="_blank">Kunjungi Google</a>
Kunjungi Google

Contoh: align (Usang)

Meskipun align sudah usang, dulunya digunakan untuk perataan teks:

<p align="center">(rata tengah) Lorem Ipsum...</p>
<p align="right">(rata kanan) Lorem Ipsum...</p>

(rata tengah) Lorem Ipsum adalah contoh teks standar dalam industri percetakan dan penataan huruf.

(rata kanan) Lorem Ipsum telah digunakan sejak tahun 1500-an.

Contoh: title

Singkatan: SD

6. Tabel

Tabel digunakan untuk menampilkan data dalam baris dan kolom. Tag utama meliputi <table>, <tr> (baris tabel), <td> (data tabel), dan <th> (header tabel).

Tabel Standar (Tanpa Header)

    <tr>
        <td>Raihan</td>
        <td>Farel</td>
        <td>Akbar</td>
    </tr>
    <tr>
        <td>Yosowilangun</td>
        <td>Lumajang</td>
        <td>Darungan</td>
    </tr>
</table>

Yosowilangun Lumajang Darungan
Tabel dengan Header (th) 
<table border="2">
    <tr>
        <th>Senin</th>
        <td>10.00</td>

    </tr>
    <tr>
        <th>Selasa</th>

        <td>12.00</td>
    </tr>

Tabel dengan rowspan dan colspan

rowspan menggabungkan sel secara vertikal, dan colspan menggabungkan sel secara horizontal.

        <td>Lari Pagi</td>
    </tr>
    <tr>
        <th rowspan="2">12.00</th>
        <td colspan="2">Makan Siang</td>
    </tr>
    <tr>
        <td colspan="2">Sholat Dhuhur</td>
    </tr>
</table>

8. Gambar

Tag <img> digunakan untuk menyematkan gambar di halaman HTML. Ini membutuhkan atribut src (sumber) dan alt (teks alternatif).

Menampilkan Gambar 
<img src="img/instagram_logo_icon_168715.png" height="100px" width="100px" alt="instagram" title="logo instagram">
Gambar dalam Teks (Perataan - Usang)

Meskipun align sudah usang, dulunya digunakan untuk mengapungkan gambar di dalam teks.

<p>
    <img src="img/instagram_logo_icon_168715.png" height="100px" width="100px" alt="instagram" align="left">
    Ini adalah paragraf dengan gambar yang sejajar ke kiri.
</p>

Ini adalah paragraf dengan gambar yang sejajar ke kiri menggunakan gaya sebaris untuk demonstrasi (atribut align sudah usang). Teks akan mengalir di sekitarnya.

9. Video

Tag <video> digunakan untuk menyematkan konten video. Atribut umum meliputi src, controls, autoplay, loop, poster, dan preload.

<video
    src="img/lv_7300184189909945607_20240404052751.mp4"
    poster="img/media_social_tiktok_icon_124256.png"
    width="100%"
    controls
    autoplay
    preload="none"
    loop>
    Browser Anda tidak mendukung tag video.
</video>

Catatan: Autoplay sering diblokir oleh browser untuk pengalaman pengguna. Teks "Browser Anda tidak mendukung tag video." muncul jika browser tidak dapat memutar video.

10. Audio

Tag <audio> digunakan untuk menyematkan konten audio. Atributnya mirip dengan video: src, controls, autoplay, loop, dan preload.

<audio
    src="web/img/KUTO KARANGANYAR TENTREM ADEM AYEM BUMI SUROKARTO - SOUND TIKTOK VIRAL 2025 [bVtGILl5nro].mp3"
    controls
    autoplay
    preload="auto"
    loop>
    Browser Anda tidak mendukung elemen audio.
</audio>

Catatan: Autoplay mungkin diblokir. Teks "Browser Anda tidak mendukung elemen audio." muncul jika browser tidak dapat memutar audio.

11. Contoh Website Multipage

Bagian ini menunjukkan bagaimana beberapa file HTML dapat dihubungkan bersama untuk membentuk situs web yang lengkap, menggunakan contoh "Perpustakaan Kabupaten Lumajang".

Ikhtisar Struktur

Situs multipage yang khas menggunakan struktur navigasi yang konsisten di semua halaman.

<!-- Navigasi Umum -->
<h4>Halaman web:</h4>
<ul>
    <li><a href="index.html">Beranda</a></li>
    <li><a href="buku.html">Daftar Buku</a></li>
    <li><a href="kontak.html">Kontak</a></li>
</ul>
Contoh: Beranda (index.html)

Halaman utama, menyambut pengguna.

<h1>Perpustakaan Kabupaten Lumajang</h1>
<hr>
